## 🔧 Customization Guide

### Colors:
Edit CSS variables in `style.css`:
```css
--primary-color: #0083b0;      /* Main blue */
--secondary-color: #00b4db;    /* Light cyan */
--accent-color: #ff6b6b;       /* Accent red */
```

### Fonts:
Currently using "Poppins" from Google Fonts. To change, modify the import in `style.css`.

### Content:
- Update company details in footer (email, phone, address)
- Customize team members in `about.html`
- Adjust pricing tiers in `services.html`
- Update FAQ in `contact.html`
